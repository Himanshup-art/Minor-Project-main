# SMC RoadMitra - Tech Stack

This document outlines the primary technologies, libraries, and services used to build the SMC RoadMitra application.

## Frontend

-   **Framework:** [Next.js](https://nextjs.org/) (with App Router)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **UI Library:** [React](https://react.dev/)

## UI & Styling

-   **Component Library:** [ShadCN/UI](https://ui.shadcn.com/)
-   **CSS Framework:** [Tailwind CSS](https://tailwindcss.com/)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Charts & Graphs:** [Recharts](https://recharts.org/)
-   **Animations:** [Tailwind CSS Animate](https://tailwindcss.com/docs/animation) & [Framer Motion](https://www.framer.com/motion/) (via ShadCN)

## Backend & Database

-   **Backend as a Service (BaaS):** [Firebase](https://firebase.google.com/)
    -   **Authentication:** Firebase Authentication (for user management and role-based access)
    -   **Database:** Cloud Firestore (as the primary NoSQL database)
-   **Deployment:** [Firebase App Hosting](https://firebase.google.com/docs/app-hosting)

## Generative AI

-   **AI Toolkit:** [Genkit](https://firebase.google.com/docs/genkit)
-   **AI Models:** [Google AI (Gemini)](https://ai.google.dev/) for damage assessment and chatbot functionality.

## Forms & Validation

-   **Form Management:** [React Hook Form](https://react-hook-form.com/)
-   **Schema Validation:** [Zod](https://zod.dev/)

## Development & Tooling

-   **Package Manager:** [npm](https://www.npmjs.com/)
-   **Environment:** [Node.js](https://nodejs.org/)
